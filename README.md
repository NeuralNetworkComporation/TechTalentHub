# TechTalentHub

**HR-автоматизация онбординга и управления отпусками для Битрикс24**

---

## 📋 ОПИСАНИЕ ПРОЕКТА

**Наименование проекта:** HR-автоматизация онбординга и управления отпусками для Битрикс24

**Краткое наименование проекта:** TechTalentHub

**Цель проекта:** Разработка веб-приложения для автоматизации процессов онбординга новых сотрудников и управления заявками на отпуск с интеграцией в Битрикс24

**Заказчик проекта:** Внутренняя разработка для Маркетплейса Битрикс24. Руководитель проекта со стороны заказчика - тимлид проекта.

**Сроки реализации проекта:** 13.01.2026 - 3.04.2026

**Бюджет проекта:** Внутренний (команда из 5 человек)

---

## 📖 ГЛОССАРИЙ

| Термин | Определение |
|--------|-------------|
| Онбординг | Процесс адаптации нового сотрудника, включающий выполнение чек-листа задач первой недели |
| Чек-лист | Список задач, которые необходимо выполнить новому сотруднику в первую неделю работы |
| Заявка на отпуск | Электронное заявление сотрудника на согласование отпуска |
| Маршрут согласования | Последовательность шагов утверждения заявки |

---

## 🎯 БИЗНЕС-ЦЕЛИ

Компании, использующие Битрикс24, сталкиваются с проблемами при ручном управлении HR-процессами. На данный момент планируется выпуск приложения в Маркетплейс Битрикс24.

Основной проблемой является отсутствие автоматизации кадровых процессов внутри экосистемы Битрикс24.

Основное пожелание заказчика:

• упорядочить данные сотрудников и процессы онбординга

• сократить время обработки заявок на отпуск

• обеспечить прозрачность статусов согласования

• уменьшить количество ошибок при ручном вводе

• повысить лояльность сотрудников за счет предоставления им личного кабинета

• привлечь больше компаний-клиентов за счет автоматизации HR-процессов
